Política de Privacidade

Última atualização: 17 de junho de 2026

1. Informações Gerais

A EnglishFlow ("nós", "nosso") opera o site englishflow.com (o "Site"). Esta página informa sobre nossas políticas de coleta, uso e divulgação de informações pessoais.

2. Coleta de Dados

Nosso site é puramente informativo e não coleta ativamente dados pessoais de visitantes. Não utilizamos formulários de cadastro, login ou captura de e-mail.

3. Serviços de Terceiros

Utilizamos os seguintes serviços que podem coletar dados anonimizados:

  • Google Fonts: Para carregamento de fontes tipográficas. O Google pode coletar seu endereço IP para este fim. Política do Google.
  • GitHub Pages / Netlify: Serviço de hospedagem que pode coletar logs de acesso (endereço IP, navegador, data/hora).

4. Cookies e Armazenamento Local

Utilizamos localStorage exclusivamente para salvar sua preferência de tema (claro/escuro). Este dado é armazenado apenas no seu navegador e não é enviado a nenhum servidor.

5. Links para o Telegram

Nosso site contém links para o aplicativo Telegram. Ao clicar, você estará sujeito à Política de Privacidade do Telegram.

6. Seus Direitos (LGPD)

De acordo com a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito de:

  • Confirmar a existência de tratamento de dados
  • Acessar seus dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a eliminação de dados

Como não coletamos dados pessoais ativamente, a maioria desses direitos se aplica apenas aos serviços de terceiros mencionados.

7. Contato

Para dúvidas sobre esta política, entre em contato pelo Telegram: @inglesemfronteiras_bot.
